

### virtualenv

```shell
pip3 install virtualenv
mkdir venv && cd venv
mkdir django2.2
# 使用系统默认python
virtualenv django2.2
# 指定python版本
virtualenv -p /usr/bin/python2.7 django2.2
cd django2.2/Scripts
activate ｜ source activate
deactivate
备注：注意区别windows和linux的区别
```
